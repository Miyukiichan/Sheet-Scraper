# SheetScraper

## Overview

Work in progress at the moment but the goal of this is to scrape URLs from a list of the same website and then add them to a google sheet, filling in existing columns as required

## Dependencies

- lxml
- Pyside6
- gspread
- oauth2client
- validators
- bs4
- requests
